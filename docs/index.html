キングスカップ！！

友達や仲間と盛り上がろう！トランプ1枚めくるたびにルールが変わる定番パーティーカードゲームアプリ。ルールは自分でカスタマイズして保存できます。

プライバシーポリシー

1. 収集する情報

本アプリ（キングスカップ！！）は、広告配信サービス「Google AdMob」を利用しています。
AdMobは以下の情報を収集することがあります：

  • 広告識別子（IDFA / AAID）
  • IPアドレス
  • デバイス情報（機種名、OSバージョン等）
  • アプリの使用状況

2. 情報の利用目的

収集した情報は、以下の目的にのみ使用します：

  • ユーザーの興味・関心に基づいた広告の配信
  • 広告効果の測定・分析

収集した情報を第三者に販売することは一切ありません。

3. トラッキングの許可（iOS）

iOS 14以降のデバイスでは、アプリ起動時にAppleのATT（App Tracking Transparency）フレームワークに基づき、トラッキングの許可をお伺いするダイアログが表示されます。
許可しない場合も、引き続きアプリをご利用いただけます（パーソナライズされない広告が表示されます）。

4. Google AdMobについて

Google AdMobのプライバシーポリシーについては、以下をご確認ください：

  • Googleのプライバシーポリシー
  • Googleの広告に関するポリシー
  • パーソナライズ広告のオプトアウト

5. 広告識別子のリセット

OS操作手順
iOS設定 → プライバシーとセキュリティ → 追跡 → 追跡を許可するアプリ でオフに設定
Android設定 → Google → 広告 → 広告IDのリセット

6. データの保存について

本アプリはユーザーがカスタマイズしたルール設定をデバイス内にのみ保存します（外部サーバーへの送信は行いません）。

7. 対象年齢

本アプリは飲み物を使ったパーティーゲームを題材としており、17歳以上を対象としています。

8. ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更後はこのページに掲載します。
